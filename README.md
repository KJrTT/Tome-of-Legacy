# Tome-of-Legacy

Tome of Legacy — это 3D RPG, созданная как духовное продолжение классических игр жанра, в частности The Elder Scrolls II: Daggerfall. Мы фокусируемся на глубине игрового мира, свободе действий и атмосферности, а не на фотореалистичной графике.

### Ключевые принципы:

Свобода исследования: Нелинейный геймплей 

Атмосфера и глубина: Мир, живущий своей жизнью

Классическая механика: Упрощенная система боя и прокачки

Повествование через находки: Сюжет раскрывается через записки, книги

### Уникальные особенности:

Система выбора класса с уникальными ветками развития

Квесты, построенные на цепочках записок (6+ взаимосвязанных записок на сюжетную линию)

Компактный, но насыщенный контентом мир
-----
# Используемые технологии

Движок: Unity 6000.3.1f1 (Вышло 9 декабря 2025)

Язык программирования: C#

Система контроля версий: Git + GitHub (В гите не будет исходного кода, потому что сам проект весит более 10гб (если включать еще Library, поэтому будут просто ссылки на несколько zip-папок))

3D Моделирование: Blender / готовые ассеты (стилизованная графика)

UI/UX: Unity UGUI

Сборка: Build Profiles (Раньше было build setting)
------
# Инструкция по запуску

1. Переходите по ссылке
```
https://github.com/KJrTT/Tome-of-Legacy/releases
```
2. Скачиваете последний архив
3. Разархивируете его
4. После этого ищите файл "Tome-of-Legacy.exe" и кликайте 2 раза.

Требования к системе:

  ОС: Windows 10/11 64-bit
  
  Процессор: Intel i5 или аналогичный AMD
  
  Оперативная память: 8 GB RAM
  
  Видеокарта: DirectX 11 compatible
  
  Место на диске: 2 GB свободного места

### Порядок показа сцен

1. Стартовая сцена:

  - Выбор класса (Воин/Маг/Лучник)

2. Основной геймплей:

  - Движение (WASD + мышь)
  
  - Боевая система (ЛКМ - атака)

3. Системы:

  - Прокачка характеристик (уровень повышается через квесты или убийство мобов)
  
  - Квестовая система (цепи из записок)

### Известные проблемы:
  - ИИ врагов может застревать на сложном рельефе
  
  - UI инвентаря требует доработки под разные разрешения
  
  - Некоторые анимации требуют полировки
